Tool description:

  • User-Agent is mostly used to set request headers in the crawler program, which is the simplest disguise for the crawler program.
  • This tool automatically generates User-Agent for different platforms, different systems and different browsers by setting different parameters
  • When the parameter is not selected, it means that no restriction is set, and the User-Agent is randomly generated. When the parameter is set, the User-Agent corresponding to the parameter is generated.
  • When there is a conflict between the set parameters, the appropriate User-Agent will not be generated, and an error message will be displayed
